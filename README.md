# jorupaza

## Youtube, Netflix와 같은 콘텐츠 플랫폼들을 콘텐츠를 관리해주는 chrome extension 

## 기간: 2023.09~2023.11

## 개발 인원: 프론트 2명, 데이터 1명, 백엔드 1명

## 맡은 역할
- BackEnd 서버 기여도 100%
- DB 설계&구현
- python selenium/BeautifulSoup 크롤링
- Swagger API 문서관리
