# JORUPAZA (크롬확장 프로그램과 머신러닝을 활용한 콘텐츠 통합 관리 서비스)
### Youtube, Netflix와 같은 콘텐츠 플랫폼의 콘텐츠를 통합 관리해주는 chrome extension 

## 개발 기간
2023.09~2023.11

## 개발 인원
-프론트: 2명
-데이터: 1명
-백엔드: 1명

## 맡은 역할
- BackEnd 서버 개발 기여도 100%
- DB 설계&구현
- Swagger API 문서관리
- selenium/BeautifulSoup 활용 크롤링
